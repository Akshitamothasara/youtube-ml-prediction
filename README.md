# 📊 Predictive Modeling of YouTube Engagement (Machine Learning Project

## Overview
This project uses machine learning to predict YouTube video views based on engagement metrics such as likes, dislikes, and comments.

## Dataset
- Source: Kaggle YouTube Trending Dataset
- Size: 6,000+ video records

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Methodology
- Cleaned and processed dataset
- Selected key features: likes, dislikes, comment_count
- Built and trained regression models (Linear Regression, Random Forest)
- Evaluated performance using R² score
- Visualized predicted vs actual values

## Results
- Achieved R² score of ~0.82
- Model demonstrates strong predictive capability for video engagement

## Visualization
Scatter plot comparing actual vs predicted views shows model accuracy and variance.

## Future Improvements
- Add more features (category, publish time)
- Use advanced models (XGBoost, Gradient Boosting)
- Perform feature engineering for better accuracy
